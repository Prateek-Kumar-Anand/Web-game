# IRON SLUG – City Warfare

A retro-style 2D browser action game inspired by classic arcade shooters. Fight through enemy-filled city levels, choose your fighter, survive waves of enemies, and achieve the highest score possible.

## Features

- Retro arcade gameplay
- Multiple playable characters
- Health and ammo system
- Score tracking
- Lives system
- Character selection screen
- Pixel-art inspired visuals
- Keyboard controls
- Runs directly in the browser

## Controls

| Key | Action |
|------|---------|
| ← → | Move |
| Z / ↑ | Jump |
| X | Shoot / Attack |
| ↓ | Crouch |
| A | Double Jump |

## How to Run

### Method 1: Directly

1. Download or clone the repository.
2. Open `iron-slug.html` in any modern web browser.
3. Start playing.

### Method 2: Local Server

```bash
python -m http.server 8000
```

Open:

http://localhost:8000
